##Tech@NYU Intranet

- Master runs on a githook

###Installation

    git clone https://github.com/TechAtNYU/intranet

    cd intranet

    npm install

    # if you don't have nodemon already
    npm install -g nodemon

###Running the Application

    # development
    npm start

    # production
    NODE_ENV=production node app.js

###Test

    npm test
